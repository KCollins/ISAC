## Introduction
{:.no_toc}
The parts cost of ISAC varies based on how many modules you're installing. This documentation is a work in progress. Most of these tools can be found at makerspaces. We conducted much of this build at [think[box]](https://thinkbox.case.edu) at CWRU.

## Tools & Materials

### Drogue
* [Sewing machine](https://case.edu/thinkbox/equipment/other-equipment/sewing-machine) 
* Scissors
* Large area for layout
* Lighter


### Sensors and Electronics
* Soldering iron
* Solder
* Hot air gun/station for electronics. (sometimes called rework station)
* No clean solder paste
* Vacuum pump
* Vacuum chamber (Recommended: [Vevor 3 gallon](https://www.vevor.com/vacuum-pump-c_11109/vevor-3-gallon-vacuum-chamber-with-pump-3-5-cfm-single-stage-vacuum-pump-stainless-steel-degassing-chamber-kit-with-tempered-glass-lid-5-ft-hose-for-stabilizing-wood-degassing-silicone-resin-p_010538846220))
* disposable pipettes (3 mL size works well)
* Hot glue gun
* 3D printer
* PVC pipe cutter (can use a saw)
* PVC cement/solvent
* Electronics tweezers
* Fine flush cutters
* Wire strippers (22-30 AWG)
* Electrical tape (for color-coding, need all 9 colors)
* micro USB cable
* Arduino Nano (or clone; [for ISP programmer](./3_build.markdown#isp-programmer))
* Pogo pins (6 individual or 2x3 grid)
